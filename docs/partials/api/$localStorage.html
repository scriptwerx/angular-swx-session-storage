$localStorage

Description

Provides a key-value (string-object) storage, that is backed by localStorage with support for expiry (in days). Objects put or retrieved from this storage are automatically serialized or deserialized by angular's toJson/fromJson.

Dependencies

  • $window
  • $location
  • $cacheFactory

Usage

$localStorage($window, $location, $cacheFactory);

Parameters

ParamTypeDetails
$windowservice

The $window service.

$locationservice

The $location service.

$cacheFactoryservice

The $cacheFactory service.

Methods

  • empty()

    Delete all data from session storage and localStorage.

    Example

    $localStorage.empty();

  • get(key)

    Get data from localStorage, will return from session cache if possible for greater performance.

    Parameters
    ParamTypeDetails
    keyString

    The key of the stored data to retrieve.

    Returns
    *

    The value of the stored data or undefined.

    Example

    $localStorage.get('myKey');

  • prefix(val)

    Overrides the default domain prefix.

    N.B. Destroys the existing cache.

    Parameters
    ParamTypeDetails
    valstring

    The string to add to the persistent data prefix.

    Example

    $localStorage.prefix('myPrefix');

  • put(key, value)

    Add data to storage

    Parameters
    ParamTypeDetails
    keystring

    The key to store the data with.

    value*

    The data to store. [@param {number} expires] (expiry in days)

    Example

    $localStorage.put('myKey', { data: 'my_data' });

  • remove(key)

    Parameters
    ParamTypeDetails
    keyString

    The key of the stored data to remove.

    Example

    $localStorage.remove('myKey');

Example

myApp.$inject = ['$localStorage'];
function myApp($localStorage) {
  // Your app
}

angular
  .module('myApp', ['swxLocalStorage']);
